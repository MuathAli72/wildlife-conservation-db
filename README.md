\# Wildlife Conservation Monitoring System



CMPE344 Database Management Systems Project - Spring 2025-2026



\## Overview

A web-based database application for tracking wildlife species, habitats, sightings, health records, and threats. Supports three user roles: Admin, Ranger, and Researcher.



\## Tech Stack

\- \*\*Frontend:\*\* HTML, Bootstrap 5, Chart.js

\- \*\*Backend:\*\* PHP

\- \*\*Database:\*\* PostgreSQL (Supabase)

\- \*\*API:\*\* Supabase REST API



\## Setup

1\. Place files in XAMPP htdocs/wildlife/

2\. Start Apache in XAMPP

3\. Open http://localhost/wildlife/index.php

4\. Login: admin\_wild / password123



\## Files

\- `config.php` - Database connection

\- `index.php` - Login page

\- `dashboard.php` - Main dashboard

\- `species.php` - Species management

\- `sightings.php` - Sightings logger

\- `reports.php` - Charts and statistics

\- `database.sql` - Full DDL, DML, and PL/SQL

\- `er\_diagram.png` - Entity Relationship Diagram

\- `REPORT.md` - Full project report
